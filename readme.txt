== Description ==

BooCommerce is a premium-grade scheduling solution for WordPress. It provides a modern, fast, and responsive booking experience for your clients.

* Manage multiple services and categories.
* Assign staff members to specific services.
* Accept payments via Stripe.
* Beautiful client dashboard and CRM.
* Highly customizable design settings.
* **Developer Scalability Engine**: Built with an extensive hook system (Actions & Filters) allowing for deep integration and customization by third-party developers.

== Installation ==

1. Upload the `boocommerce` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure your settings in the BooCommerce menu.
